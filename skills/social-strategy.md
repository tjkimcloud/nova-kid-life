## Twitter/X Algorithm — NovaKidLife Optimization

### How Twitter ranks content (applies to our posts)

**Real-graph** — Twitter predicts which followers will engage. Build this by:
- Posting consistently (3–5x/week minimum)
- Replying to parents who comment on our posts
- Retweeting/quoting local NoVa news accounts (ARLnow, Reston Now, FFXnow, Patch)
- Following and engaging with @FairfaxCounty, @LoudounGov, @ArlingtonVA

**SimClusters** — Topic communities. Our target clusters:
- Northern Virginia family/parenting
- Pokémon TCG / card game community
- NoVa local news and events
- Use cluster-native language in posts (don't say "event", say "happening this weekend")

**TwHIN** — User-content relevance. Establish NovaKidLife as the authoritative source for:
- "things to do with kids in Northern Virginia"
- "free events this weekend NoVa"
- "Pokémon TCG NoVa"

**Tweepcred** — Credibility signals:
- Never delete tweets (lowers score)
- Don't post identical copy across events
- Always include the event link (signals content value)
- Avoid engagement bait ("RT if you're a parent!")

### Twitter post formula
```
[Hook — what + when] → [Specifics — location, cost, age] → [CTA + link] → [2–3 hashtags]
```

**Example:**
```
Free storytime this Saturday at Fairfax County Library 📚
Ages 2–6 | 10:00 AM | Chantilly branch
No registration needed → novakidlife.com/events/[slug]
#NoVaKids #FairfaxVA #FreeKidsEvents
```

### Negative signals to avoid
- Never post only an image with no text context
- Never use more than 4 hashtags
- Never repost the same event copy twice
- Avoid "like and share" language

## Instagram Strategy

### Content pillars (rotate through these)
1. **Event announcements** — event image + date/time/location in caption, link in bio
2. **Weekend roundup** — carousel of 3–5 upcoming events ("5 Free Things This Weekend in NoVa")
3. **Pokémon TCG** — card set drops, league night alerts, store spotlights
4. **Deal alerts** — restaurant/activity deals with the offer clearly stated
5. **Seasonal highlights** — pumpkin patches, holiday events, summer camps

### Caption structure
```
[Headline — what + when]
[2–3 lines of key details: location, age, cost]
[CTA — "Link in bio for full details and registration"]
.
.
[Hashtags — 10–15, mix of broad + local]
```

### Instagram hashtag playbook
**Local (always use):**
`#NorthernVirginia #NoVaKids #FairfaxVA #LoudounCounty #ArlingtonVA #RestonVA`

**Topic (match to event):**
`#KidsActivities #FamilyFun #FreeKidsEvents #StorytimeKids #STEMKids #PokemonTCG #PokemonCards`

**Seasonal (add when relevant):**
`#FallFun #SummerKids #HolidayEvents #PumpkinPatch`

### Instagram Reels (high-reach format)
- Use for seasonal events (Cox Farms, Meadowlark, Wolf Trap summer)
- 15–30 seconds, vertical video
- Text overlays: event name, date, cost, location
- Audio: trending sounds or simple upbeat music
- No filming required — use event's own promotional images with Ken Burns effect

### Stories (time-sensitive)
- Post event reminder Stories 24–48 hours before event
- Use countdown sticker (drives algorithm engagement)
- Poll sticker: "Are you going this weekend? 👍/👎" (engagement signal)
- Swipe-up / link sticker to event page

## Facebook Strategy

### Groups > Pages
The highest-value Facebook activity is participation in local groups, not our own page.

**Target groups to monitor and share in:**
- Northern Virginia Moms / Dads groups
- Fairfax County Family Events
- Loudoun County Parents
- Reston & Herndon Families
- Vienna/McLean Parents
- Northern Virginia Pokémon TCG players

**Posting rules for groups:**
- Always add genuine value — don't just drop a link
- Include the key details in the post (don't force them to click)
- Respond to comments within 24 hours
- Never post the same content to multiple groups same day

### Facebook Page post formula
```
[Emoji + Hook]
[Date, time, location, cost on separate lines]
[1 sentence description]
[Link to event page]
[1–2 hashtags]
```

**Example:**
```
🎃 Fall Farm Fun at Cox Farms this weekend!

📅 Sat–Sun, Oct 18–19
⏰ 9:00 AM – 6:00 PM
📍 15621 Braddock Rd, Centreville VA
💰 $20/person, under 2 free

Pick pumpkins, meet animals, hay rides and more.
Full details + tickets → novakidlife.com/events/cox-farms-fall-2026

#NoVaKids #FallFun
```

## Content Calendar Framework

### Weekly cadence (automated via Buffer)
| Day | Platform | Content Type |
|-----|----------|-------------|
| Monday | Twitter/X | "This week's top picks" thread — 5 events |
| Tuesday | Instagram | Single event announcement |
| Wednesday | Facebook | Deal alert or free event highlight |
| Thursday | Twitter/X | Pokémon TCG event/drop alert |
| Friday | Instagram | "Weekend Roundup" carousel |
| Saturday | Twitter/X | Day-of reminder for top event |
| Sunday | Facebook | Next week preview |

### Priority triggers (post immediately regardless of cadence)
- New Pokémon TCG set release date confirmed
- Rare free event at a major venue (Wolf Trap, Meadowlark)
- Time-sensitive deal (expires soon)
- Breaking local event cancellation
