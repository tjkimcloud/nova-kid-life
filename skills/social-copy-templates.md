# Skill: Social Copy Templates

**Reference:** Use when writing or generating social posts. For platform strategy and content calendar, see `social-strategy.md`. For hashtag lists, see `social-hashtags.md`.

---

## Twitter/X Post Formula

```
[Hook — what + when] → [Specifics — location, cost, age] → [CTA + link] → [2–3 hashtags]
```

**Example:**
```
Free storytime this Saturday at Fairfax County Library 📚
Ages 2–6 | 10:00 AM | Chantilly branch
No registration needed → novakidlife.com/events/[slug]
#NoVaKids #FairfaxVA #FreeKidsEvents
```

**Negative signals — never do this:**
- More than 4 hashtags
- Only an image with no text context
- Reposting the same event copy twice
- "Like and share" language

## Instagram Caption Structure

```
[Headline — what + when]
[2–3 lines of key details: location, age, cost]
[CTA — "Link in bio for full details and registration"]
.
.
[Hashtags — 10–15, mix of broad + local — see social-hashtags.md]
```

---

## Facebook Page Post Formula

```
[Emoji + Hook]
[Date, time, location, cost on separate lines]
[1 sentence description]
[Link to event page]
[1–2 hashtags]
```

**Example:**
```
🎃 Fall Farm Fun at Cox Farms this weekend!

📅 Sat–Sun, Oct 18–19
⏰ 9:00 AM – 6:00 PM
📍 15621 Braddock Rd, Centreville VA
💰 $20/person, under 2 free

Pick pumpkins, meet animals, hay rides and more.
Full details + tickets → novakidlife.com/events/cox-farms-fall-2026

#NoVaKids #FallFun
```

## Copy Variations by Event Type

### Standard family event
```
[Platform-appropriate emoji] [Title]
📅 [Weekday, Month Date]
⏰ [Time]
📍 [Venue, City]
[Free/Cost indicator]

[1-sentence hook]

[Link] [hashtags]
```

### Deal / freebie alert
```
🚨 DEAL: [What the deal is]
Valid: [timeframe]
📍 [Location/online]

[Any restrictions or promo code]

Full details → [link]
[hashtags]
```
