# Local SEO Monthly Audit Checklist — NovaKidLife

Standalone reference file for `local-seo-tracking.md`. Run every month.

---

## Monthly Audit Checklist

### GBP
- [ ] 4+ GBP posts published this month (cross-reference `local-seo-gbp.md` for post format)
- [ ] All reviews responded to within 24h
- [ ] New photos added (2+/month)
- [ ] Q&A checked for new questions — seed answers immediately

### Citations
- [ ] No new NAP inconsistencies (check BrightLocal)
- [ ] Any new directories discovered? Submit with exact NAP format

### Rankings
- [ ] Run all 20 target queries in BrightLocal (query list in `local-seo-tracking.md`)
- [ ] No significant drops (investigate if >5 position drop on any tracked query)

### Search Console
- [ ] No new coverage errors
- [ ] Review new queries appearing in GSC — any new keyword clusters to target?
- [ ] CTR below 2% on any page? → Rewrite title/meta (use `seo-title-optimizer.md`)

### Backlinks
- [ ] New links acquired this month?
- [ ] Any new outreach sent? (use templates in `local-seo-content.md`)

### Blog
- [ ] At least 2 local blog posts published
- [ ] Any seasonal content due for update?

### Social
- [ ] Weekly GBP posts maintained
- [ ] Weekly Nextdoor posts maintained
- [ ] Any Facebook group shares this month?

### Analytics
- [ ] Any city driving unexpected traffic? Double down on content for that city
- [ ] Any event type with high register_click rate? Prioritize that category in blog content

### LLM Citation Test (see `llm-citation-targets.md`)
- [ ] Tested all 10 target prompts on all 5 platforms
- [ ] Results logged
- [ ] Any new citations found? Add that source to co-citation building list
