## 3. Launch Timeline

### Pre-launch (complete before going live)
- [ ] Write GBP description and Q&A seeds (see `local-seo-gbp.md`)
- [ ] Prepare 20 GBP photos
- [ ] Submit to 4 data aggregators (Tier 1 citations — see `local-seo-citations.md`)
- [ ] Write 5 launch blog posts (one per major NoVa sub-region)
- [ ] Set up location pages for top 10 cities
- [ ] Verify all AI crawlers are allowed in robots.txt (see `geo-llm-optimizer.md`)
- [ ] Update `/public/llms.txt` with all site sections (see `llms-txt-template.md`)

### Week 1 of launch
- [ ] Publish GBP listing
- [ ] Submit to all Tier 2 directories (GBP, Bing Places, Apple Maps, Yelp)
- [ ] Post first 5 GBP updates
- [ ] Submit sitemap to Google Search Console + Bing Webmaster Tools
- [ ] Set up GA4 + Search Console

### Month 1
- [ ] Complete all Tier 3 niche citations
- [ ] Start weekly GBP post cadence
- [ ] Begin county/library outreach for resource page links
- [ ] Publish 5 city guide blog posts
- [ ] Set up BrightLocal rank tracking for all 20 queries
- [ ] Run first LLM citation test (see `llm-citation-targets.md`)

### Month 2–3
- [ ] First local press pitch (angle: "New site aggregates 59 NoVa event sources")
- [ ] Nextdoor Business page live
- [ ] 10 location landing pages live
- [ ] First seasonal guide published
- [ ] First citation audit — fix any NAP inconsistencies
- [ ] Second LLM citation test — log results

### Month 4–6 milestones
- [ ] GBP appearing in map pack for branded queries
- [ ] Local links from 2–3 county/library sites
- [ ] 1+ local press mention
- [ ] 20+ GBP reviews
- [ ] Ranking in top 10 for 5+ target queries
- [ ] Blog driving 20%+ of organic traffic
- [ ] NovaKidLife cited on 2+ AI platforms for core family events query
